# 任氏外卖微信小程序 - 项目实施计划书

## 项目概述

**任氏外卖** 是一个基于微信小程序的 AI 点餐系统，采用 Spring Boot 3.2.0 后端 + 微信小程序前端的架构。系统集成 AI 智能客服（ModelScope Qwen3-1.7B 模型），支持菜单浏览、购物车、订单管理等功能。
