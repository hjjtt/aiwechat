## 数据库ER图

```mermaid
erDiagram
    USERS ||--o{ USER_SESSIONS : has
    USERS ||--o{ ORDERS : places
    USERS ||--o{ CHAT_RECORDS : creates
    USERS ||--o{ USER_FAVORITES : has
    USERS ||--o{ USER_ADDRESSES : has
    ORDERS ||--o{ ORDER_ITEMS : contains
    MENUS ||--o{ ORDER_ITEMS : included_in
    MENUS ||--o{ USER_FAVORITES : favorited_by

    USERS {
        bigint id PK
        varchar open_id UK
        varchar nickname
        varchar avatar_url
        varchar role
        varchar status
        datetime created_at
        datetime last_login_at
    }

    USER_SESSIONS {
        bigint id PK
        bigint user_id FK
        varchar token UK
        datetime expires_at
        datetime last_active_at
    }

    ORDERS {
        bigint id PK
        varchar order_number UK
        bigint user_id FK
        varchar status
        decimal total_amount
        varchar contact_name
        varchar contact_phone
        varchar delivery_address
        varchar remark
        datetime created_at
        datetime updated_at
    }

    ORDER_ITEMS {
        bigint id PK
        bigint order_id FK
        bigint menu_id FK
        varchar menu_name
        int quantity
        decimal unit_price
        decimal subtotal
    }

    MENUS {
        bigint id PK
        varchar name
        text description
        decimal price
        varchar category
        varchar image_url
        boolean is_available
        int sales_count
    }

    CHAT_RECORDS {
        bigint id PK
        bigint user_id FK
        varchar session_id
        text question
        text answer
        varchar role
        datetime created_at
    }

    USER_FAVORITES {
        bigint id PK
        bigint user_id FK
        bigint menu_id FK
        datetime created_at
    }

    USER_ADDRESSES {
        bigint id PK
        bigint user_id FK
        varchar contact_name
        varchar phone
        varchar address
        decimal latitude
        decimal longitude
        boolean is_default
        datetime created_at
    }
```

## RAG知识检索流程图

```mermaid
flowchart TD
    A[用户问题] --> B[问题向量化]
    B --> C[向量数据库检索]
    C --> D{相似度阈值检查}
    D -->|相似度>0.7| E[获取Top-K结果]
    D -->|相似度<0.7| F[使用通用回答]
    E --> G[构建Prompt]
    F --> H[直接生成回答]
    G --> I[调用LLM生成回答]
    I --> J[返回回答]
    H --> J
```
